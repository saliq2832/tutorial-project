Array operations in JavaScript



an array is of the form a[]
it can include any type of data types

  • push()

    to add element at the last of the array
    eg: let arr=[1,2,3]
    arr.push(8);

    output: arr=[1,2,3,8] .

  • pop()

    to remove an element from last of the array
    eg: let arr=[1,2,3]
    arr.pop();

    output: arr=[1,2] .

  • shift()

    to remove an element at the front of the array
    eg: let arr=["hi",2,3]
    arr.shift();

    output: arr=[2,3] .

  • unshift()

    to add an element at the front of the array
    eg: let arr=["hi",2,3]
    arr.shift(4);

    output: arr=[4,"hi"2,3] .
